# Vijays ChatApp (React + Firebase)

This is a starter React + Firebase chat app scaffold prepared for GitHub Pages deployment at:
https://vijayoliver.github.io/vijays-chatapp-react/

## Quick local run
1. `npm install`
2. Replace `src/firebase.js` placeholders with your Firebase project's config.
3. `npm run dev`

## To deploy to GitHub Pages
1. Ensure repo name: `vijays-chatapp-react`
2. `npm install --save-dev gh-pages`
3. `npm run deploy`

See full instructions in the provided files.
